Winter of Code 1.0 is Cyber Labs' first Winter Hackathon.

Winter of Code is a platform where technological enthusiasts come together and contribute to tools that solve real-life problems. With the efforts of Cyber Labs team, WoC is happening for the first time in IIT(ISM) Dhanbad and we are proud to announce the first Winter Open Source Hackathon of IIT(ISM) Dhanbad.

We are all set to make Winter of Code 1.0 a great success.
